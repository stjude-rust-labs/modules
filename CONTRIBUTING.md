# Contributing

Contributions of new modules and improvements to existing ones are welcome.
Before opening a pull request, read the [licensing](README.md#licensing)
section—in particular the [tool license requirements](README.md#tool-licenses)
and the [expectations for contributors](README.md#expectations-for-contributors).
A pull request template is included to guide you through these requirements.

For questions and discussion, join the OpenWDL community on
[Slack](https://join.slack.com/t/openwdl/shared_invite/zt-ctmj4mhf-cFBNxIiZYs6SY9HgM9UAVw).
